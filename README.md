# Aniworld Extension

A browser extension that enhances your Aniworld streaming experience.

## Overview

Aniworld Extension is a browser extension designed to improve the functionality and user experience when using the Aniworld streaming platform. This extension adds various quality-of-life features and enhancements to make your anime watching experience more enjoyable.

## Features

- 🎬 Enhanced video playback controls
- ⚡ Improved streaming performance
- 🎨 Customizable interface options
- 📱 Cross-browser compatibility
- 🔒 Privacy-focused design

## Installation

### From Source

1. Clone this repository:
   ```bash
   git clone https://github.com/SaufGott/Aniworld-extension.git
   cd Aniworld-extension
   ```

2. Load the extension in your browser:

   **Chrome/Edge:**
   - Open `chrome://extensions/` (or `edge://extensions/`)
   - Enable "Developer mode" (toggle in the top right)
   - Click "Load unpacked"
   - Select the repository folder

   **Firefox:**
   - Open `about:debugging#/runtime/this-firefox`
   - Click "Load Temporary Add-on"
   - Select any file in the repository folder

## Usage

After installing the extension, simply visit [Aniworld](https://aniworld.to) and the extension will automatically enhance your experience.
